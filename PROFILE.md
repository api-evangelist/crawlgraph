# CrawlGraph

A backlink-intelligence SaaS built on Common Crawl's open hyperlink graph (~120M domains / 4.4B links). Provides referring-domain lookups, competitor gap analysis, and outreach-target discovery as a low-cost alternative to Ahrefs/Moz/Semrush.

- **Provider:** https://crawlgraph.com
- **Documentation:** https://crawlgraph.com/docs/api
- **Base URL:** https://crawlgraph.com/api/v1/
- **Tags:** SEO, backlink-intelligence, martech, competitive-intelligence, web-data, Common-Crawl, link-building, developer-tools, API, MCP
- **Public API:** yes
- **Agent-native (MCP / llms.txt / skills):** yes
- **Confidence:** 85/100

## APIs

- **CrawlGraph REST API v1** — JSON REST API for backlink/referring-domain lookups, indexed Common Crawl release listing, and async competitor gap analysis. Bearer auth (cg_live_ keys); API access requires the $99 lifetime tier. (`https://crawlgraph.com/api/v1/`)

> REST API base https://crawlgraph.com/api/v1/ with Bearer auth (cg_live_ keys); full usage requires a $99 lifetime key. An OpenAPI 3.1 schema is documented at https://crawlgraph.com/api/v1/openapi.json but sits under the auth-gated /api/v1/* namespace and public unauthenticated retrieval could NOT be confirmed, so it is omitted from properties. The crawlgraph-mcp MCP server (github.com/pucilpet/crawlgraph-mcp, npm/MIT, 4 tools: backlinks, gap_analysis, gap_outreach_targets, releases) is a LOCAL stdio server run via 'npx -y crawlgraph-mcp' — there is NO hosted/remote MCP endpoint, so it is not cataloged as an MCP property. agent_native=true is based on the confirmed public llms.txt. No published agent skills found. The /apis.json path returns a backlink-report page rather than a real APIs.json document. Related open datasets at github.com/pucilpet/crawlgraph-datasets (CC-BY) are data, not an interface.
